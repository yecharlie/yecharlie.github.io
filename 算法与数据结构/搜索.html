基本方法

深度优先搜素

深度优先搜索约定先对自己做访问标记，再对邻居节点依次调用深度优先搜索。

非递归实现深度优先搜索，利用栈实现，注意有两处要做访问标记，另外注意如何利用locals顺时针访问四个方向上的邻居。

class Solution {
    using ipair = pair<int, int>;
    const vector<int> locals={1,0,-1,0,1}; // clockwise handle
public:
    int maxAreaOfIsland(vector<vector<int>>& grid) {
        int m = grid.size(), n = m > 0 ? grid[0].size() : 0;
        if (!m*n) return 0;

        int res = 0;
        for (int r = 0; r < m; r ++)
            for (int c = 0; c < n; c ++) {
                if (grid[r][c] != 1) continue;

                grid[r][c] = 2;
                stack<ipair> s;s.push({r,c});   
                int area = 0;
                while (s.size()) {
                    auto p = s.top(); s.pop();
                    area ++;

                    for (int lc = 0; lc < 4; lc ++) {
                        int lcx = p.first + locals[lc], lcy = p.second + locals[lc+1];

                        if (lcx >= 0 && lcx < m &&
                            lcy >= 0 && lcy < n &&
                            grid[lcx][lcy] == 1) {
                                grid[lcx][lcy] = 2; // locate
                                // psuh the locals
                                s.push({lcx, lcy});
                            }
                    }
                }

                if (area > res) res = area;
            }
            return res;
    }
};

回溯法

带状态的深度优先搜索，常用于排列、组合、选择的问题，在回溯的过程要同步修改状态。

例 带重复元素的组合，这里注意几点：

  1. 递归函数签名：

    combinationSum2([output]…[origin]…[aux]…[changed arr]…[changed num])

  2. 组合方式，\(Comb_i\)每次从[i,n]选出一个有效元素j，然后继续调用\(Comb_{j+1}\)

  3. 防止重复，首先对候选数排序，只要保证在同一级上选择不重复，结果就不会重复。

定理1 如果可能的组合数都是按递增顺序产生的，那么在组合的过程中，假设从第K（\(K \ge 1\)）次开始出现了选择分岔，由此引出的两个状态空间的解的交集也为空。（证明略）

现在证明该算法每个解的唯一性。

容易证明：在第K（\(K \ge 1\)）层递归里，第27行里的candidates[j]每次都不同，根据定理1，它们对应的解彼此没有交集；于是转而考察其中任意存在解的子状态空间的内部情况，如果这个子空间已经是一个解（终止式），显然这个解是独立的，否则令K=K+1，再重复利用上部的结论；因为n有限，\(K \le n\)，于是终止式可达，原问题得证。

class Solution {
public:
    vector<vector<int>> combinationSum2(vector<int>& candidates, int target) {
        vector<vector<int>> res;
        int n = candidates.size();
        if (!n) return res;

        vector<int> p;
        sort(candidates.begin(), candidates.end());
        combinationSum2(res, candidates, n, p, target, 0);
        return res;
    }

    void combinationSum2(vector<vector<int>>& res,
        vector<int>& candidates, int n, vector<int>& p, int target, int i) {
        if (!target) {
            res.push_back(p);
            return;
        } else if (i >= n || candidates[i] > target) {
            return;
        }

        int pre = candidates[i]-1;
        for (int j = i; j < n; j ++)
            if (candidates[j] != pre) {
                p.push_back(candidates[j]);
                combinationSum2(res, candidates, n, p, target-candidates[j], j+1);
                p.pop_back();

                pre = candidates[j];
            }
    }
};
